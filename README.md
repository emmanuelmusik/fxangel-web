# 👼 FXAngel Web App

This is the FXAngel frontend. Deploy to Vercel in 3 steps.

## Deploy to Vercel

1. Upload this entire folder to a new GitHub repository called `fxangel-web`
2. Go to vercel.com → New Project → select `fxangel-web`
3. Click Deploy — done!

Your app will be live at: https://fxangel.vercel.app (or similar)
